## 🎨 디자인 가이드라인

### 컨셉
**눈치코치(Sense Coach)**의 핵심 가치를 시각적으로 표현:
- 학교 / 교육
- AI / 기술
- 따뜻함 / 도움
- 해외 / 글로벌

### 색상 팔레트

**주요 색상** (앱의 Primary Color):
```
#FFB6C1 - Light Pink (파스텔 핑크)
```

**보조 색상**:
```
#FFC0CB - Pink (연한 핑크)
#FFD700 - Gold (골드, 강조용)
#4A90E2 - Blue (파란색, 신뢰감)
#FFFFFF - White (배경)
```

**추천 조합**:
- 배경: White 또는 Light Pink
- 주요 요소: Pink + Gold
- 강조: Blue

### 아이콘 스타일

**1. 미니멀 (권장)**
- 간결하고 깔끔한 디자인
- 복잡하지 않은 형태
- 작은 크기에서도 알아보기 쉬움

**2. 플랫 디자인**
- 그림자나 그라데이션 최소화
- 단순한 색상 블록
- 현대적인 느낌

**3. 아이소메트릭 (선택사항)**
- 3D 느낌의 2D 아이콘
- 독특하고 눈에 띔
- 약간 더 복잡할 수 있음

## 💡 아이콘 디자인 아이디어

### 옵션 1: 책가방 + AI
```
┌─────────────┐
│             │
│   🎒 + 🤖   │  책가방 + AI 로봇 조합
│             │
│  Sense      │  작은 텍스트 (선택)
│  Coach      │
└─────────────┘
```
- **장점**: 학교(책가방) + AI를 명확히 표현
- **색상**: 핑크 책가방, 파란색 AI 요소

### 옵션 2: 알림장 + 돋보기
```
┌─────────────┐
│             │
│  📋 + 🔍    │  문서 + 돋보기
│             │
│     SC      │  이니셜 (Sense Coach)
└─────────────┘
```
- **장점**: '분석'의 의미 강조
- **색상**: 핑크 배경, 흰색 문서, 골드 돋보기

### 옵션 3: 지구본 + 하트
```
┌─────────────┐
│             │
│   🌍 + ❤️   │  글로벌 + 따뜻함
│             │
│    눈치      │  한글 또는 영문
│    코치      │
└─────────────┘
```
- **장점**: 해외 거주 + 따뜻한 도움을 표현
- **색상**: 파란색 지구본, 핑크 하트

### 옵션 4: 체크리스트 + 별
```
┌─────────────┐
│             │
│   ✅ + ⭐   │  완료 + 우수함
│             │
│     SC      │
└─────────────┘
```
- **장점**: 체크리스트 기능 강조
- **색상**: 핑크 배경, 흰색 체크마크, 골드 별

```

│             │
│             │
│     SC      │  Sense Coach
│             │
```

- **색상**: 그라데이션 배경 (핑크→파란색)

## 📏 디자인 규칙

### iOS 가이드라인
1. **정사각형**: 정확히 1024x1024px
2. **투명도 없음**: 배경은 불투명
3. **둥근 모서리 없음**: iOS가 자동으로 처리
4. **알파 채널 없음**: RGB only
5. **텍스트 최소화**: 아이콘 자체로 의미 전달
6. **색상 프로파일**: sRGB 또는 Display P3

### Android 가이드라인
1. **정사각형**: 512x512px (Play Store)
2. **적응형 아이콘**: Foreground + Background 분리 권장
3. **투명도 가능**: PNG 알파 채널 지원
4. **안전 영역**: 중앙 66%만 사용 (나머지는 잘릴 수 있음)

### 공통 가이드라인
1. **단순함**: 복잡한 디테일 피하기
2. **대비**: 배경과 요소의 명확한 구분
3. **확장성**: 작은 크기에서도 알아보기 쉬움
4. **브랜드 일관성**: 앱 내 디자인과 조화
5. **경쟁사 분석**: 유사 앱과 차별화

### 수동 리사이징

#### ImageMagick (CLI 도구)
```bash
# 설치
brew install imagemagick  # macOS
sudo apt install imagemagick  # Linux

# 모든 크기 생성 스크립트
for size in 1024 512 192 180 167 152 144 120 96 87 80 76 72 60 58 48 40 29 20; do
  convert icon_original.png -resize ${size}x${size} icon_${size}.png
done
```

#### Python 스크립트
```python
from PIL import Image

sizes = [1024, 512, 192, 180, 167, 152, 144, 120, 
         96, 87, 80, 76, 72, 60, 58, 48, 40, 29, 20]

original = Image.open('icon_1024.png')

for size in sizes:
    resized = original.resize((size, size), Image.LANCZOS)
    resized.save(f'icon_{size}.png')
```

## 🚀 제출 방법

### iOS App Store
1. Xcode의 Asset Catalog에 추가
2. 또는 App Store Connect에서 직접 업로드
3. "App Icon" 섹션에서 1024x1024 업로드

### Google Play Store
1. Play Console → "그래픽 자산"
2. "앱 아이콘" 섹션에서 512x512 업로드
3. Android Studio의 `res/` 폴더에 다양한 크기 추가
