# 이미지 디자인 분석 및 CSS 가이드

## 📊 이미지 디자인 요소 분석

### 1. 컬러 팔레트 (Color Palette)

**주요 배경 색상:**
- **라이트 피치 (Light Peach)**: `#FFE5D9` - 부드럽고 따뜻한 톤
- **라이트 그린 (Light Mint)**: `#D4EDDA` - 신선하고 편안한 톤
- **라이트 퍼플 (Light Lavender)**: `#E9D5FF` - 우아하고 차분한 톤
- **라이트 베이지 (Light Beige)**: `#F5F0E8` - 중성적이고 고급스러운 톤
- **크림 화이트 (Cream White)**: `#FEFCF9` - 메인 배경

**텍스트 색상:**
- **다크 그레이 (Dark Gray)**: `#2D3748` - 주요 텍스트
- **미디엄 그레이 (Medium Gray)**: `#4A5568` - 보조 텍스트
- **라이트 그레이 (Light Gray)**: `#718096` - 힌트/플레이스홀더

**강조 색상 (Accent Colors):**
- **옐로우 (Yellow)**: `#FBD38D` - 중요한 요소
- **레드 (Red)**: `#FC8181` - 시간 표시 등
- **다크 퍼플 (Dark Purple)**: `#553C9A` - 선택된 상태

### 2. 레이아웃 (Layout)

**특징:**
- 카드 기반 그리드 시스템
- 넓은 여백과 패딩 (16-24px)
- 수평 스크롤 가능한 카테고리 섹션
- 타임라인 스타일의 태스크 리스트
- 고정 헤더 + 스크롤 콘텐츠

**간격 시스템:**
- 작은 간격: 8px
- 기본 간격: 16px
- 큰 간격: 24px
- 섹션 간격: 32px

### 3. 타이포그래피 (Typography)

**폰트 패밀리:**
- Sans-serif (Inter 또는 시스템 기본 폰트)
- 한국어: Noto Sans KR, Apple SD Gothic Neo

**폰트 크기:**
- 헤더 (H1): 28-32px, Bold (700)
- 서브헤더 (H2): 20-24px, Semi-Bold (600)
- 본문 (Body): 16px, Regular (400)
- 보조 텍스트 (Caption): 14px, Regular (400)
- 작은 텍스트 (Small): 12px, Regular (400)

**폰트 웨이트:**
- Light: 300
- Regular: 400
- Medium: 500
- Semi-Bold: 600
- Bold: 700

**라인 하이트:**
- 헤더: 1.2-1.3
- 본문: 1.5-1.6
- 보조 텍스트: 1.4

### 4. 카드 스타일 (Card Style)

**특징:**
- **둥근 모서리**: 16-24px border-radius
- **부드러운 그림자**: `0 4px 12px rgba(0, 0, 0, 0.08)`
- **미묘한 배경 텍스처**: 그라디언트 또는 텍스처 패턴
- **아이콘 통합**: 왼쪽/오른쪽에 아이콘 배치
- **아바타 그룹**: 작은 원형 아바타 나열
- **진행률 인디케이터**: 원형 또는 선형 진행률 표시

**카드 유형:**
1. **카테고리 카드**: 큰 카드, 일러스트 포함, 색상 배경
2. **태스크 카드**: 작은 카드, 정보 밀집, 부드러운 배경
3. **타임라인 카드**: 시간 표시 포함, 세로 리스트

## 🎨 CSS 변환 가이드

### 적용 전략

1. **컬러 팔레트 교체**: 현재 그라디언트 → 파스텔 톤
2. **카드 스타일 개선**: 더 부드럽고 둥근 스타일
3. **타이포그래피 조정**: 크기와 웨이트 최적화
4. **레이아웃 개선**: 여백과 간격 조정
5. **아이콘 통합**: SVG 아이콘 스타일 개선

### 우선순위

1. ✅ 컬러 팔레트 (최우선)
2. ✅ 카드 스타일
3. ✅ 타이포그래피
4. ✅ 그림자 효과
5. ⚠️ 레이아웃 구조 (Streamlit 제약 고려)
