# ✅ 앱스토어 등록 체크리스트 (App Store Submission Checklist)

이 문서는 눈치코치(Sense Coach) 앱을 iOS App Store와 Google Play Store에 등록하기 위한 단계별 체크리스트입니다.

---

## 📋 사전 준비 단계

### 1. 개발자 계정
- [ ] Apple Developer Program 가입 완료 ($99/년)
  - [ ] 결제 정보 등록
  - [ ] 계약 동의
- [ ] Google Play Console 가입 완료 ($25 일회성)
  - [ ] 결제 정보 등록
  - [ ] 개발자 계정 인증

### 2. 클라우드 배포
- [ ] Streamlit 앱 클라우드 배포 완료
  - [ ] Streamlit Cloud 또는 AWS/GCP 선택
  - [ ] 도메인 연결 (선택사항)
  - [ ] HTTPS 설정 완료
  - [ ] API 키 환경 변수 설정
- [ ] 앱 URL 테스트 및 정상 작동 확인

### 3. 법적 문서
- [ ] 개인정보 처리방침 작성 및 게시
  - [ ] 웹사이트 또는 GitHub Pages에 업로드
  - [ ] URL 확보: `https://your-domain.com/privacy-policy`
- [ ] 서비스 이용약관 작성 (선택사항)
- [ ] 연락처 정보 준비
  - [ ] 지원 이메일: `support@sensecoach.app`
  - [ ] 웹사이트 URL
